This function returns a pointer to vector<Point3f> given a mask image, only the non zero values are taken into account, and a probability image from a matching previous step.

Returns
void
Parameters
imgA Mat containing an image with 0's and 1's, it is the mask image, it has H x W dimensions.
probA Mat containing probability values in each pixel, it has H' x W' dimensions.
*pointsA pointer to vector<Point3f>, it is the output parameter.
pdiffA point with the required shift in x and y coordinates to bring the point from prob to img coordinates.
scaleA double for the current scale, it is used to bring the calculated coordinates to the original image size.
Precondition
The images img and prob do not have the same dimensions.
